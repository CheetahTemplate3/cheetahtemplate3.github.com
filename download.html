Downloading Cheetah¶

The most recent stable release of Cheetah v2.2.1 which was released on June 1st, 2009.

Download v2.2.1¶

  • v2.2.1 tarball
  • v2.2.1 zip

You can keep up to date with release candidates or other downloads of Cheetah by visiting the cheetahtemplate GitHub page

Note to Windows users: You should install the compiled version of Cheetah’s NameMapper. It is dramatically faster than the pure Python version, in the source tree you can find compiled versions for Python 2.4: namemapper.pyd2.4, Python 2.5: namemapper.pyd2.5. Install it wherever your system puts Cheetah/NameMapper.py. Strip the version numbers off the filename.
